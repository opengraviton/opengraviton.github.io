The Gravity-Defying Architecture

🗜️

Extreme Quantization

Features state-of-the-art 1.58-bit Ternary quantization, collapsing 16-bit weights down to just {-1, 0, +1} for absolutely massive 10x compression ratios.

⚡

Dynamic Sparsity

Replaces dense computations with Top-K zeroing and Mixture of Experts (MoE) routing. Dynamically prunes 70%+ of the compute pipeline per token natively.

💿

Layer Streaming

Limits shattered. Bypasses physical RAM limitations by memory-mapping (mmap) weights asynchronously directly from NVMe SSDs to the computation engine.

🧠

Speculative Decoding

Accelerates generation by 2-3x by utilizing Draft vs Target generation heuristics, bypassing memory bandwidth walls during autoregressive decoding.

🔩

QuantizedLinear

Drop-in replacement for nn.Linear that stores weights in packed quantized format. INT8 saves 62% memory with near-zero quality loss. Mixed-precision routes critical layers to 8-bit, FFN to 4-bit automatically.

🖥️

Graviton UI

Beautiful dark-themed chat interface — enter a HuggingFace token, pick a model and quantization mode, and chat with real-time SSE streaming and live tok/s counter. One command: graviton-ui.

🧪

88 Tests, Full Coverage

Every component is battle-tested: attention masks, quantizer device consistency, speculative decoding rollback, KV cache fast-path and compressed modes, and end-to-end model inference — all validated in under 3 seconds.

Initialize in Seconds

bash 

# 1. Clone and install Graviton
git clone https://github.com/opengraviton/graviton.git
cd graviton
pip install -e ".[all]"

# 2. View your hardware capabilities
python3 -m graviton.cli.main info

# 3. Run a model locally (CLI)
graviton run TinyLlama/TinyLlama-1.1B-Chat-v1.0 \
    -p 'Explain quantum computing in one sentence:'

# Or launch the Web UI
pip install graviton-ui
graviton-ui          # opens http://localhost:7860
output 
Quantized 154 linear layers, saved 1318 MB in packed storage
Model ready: 0.13B params, 0.78 GB on mps
Quantization: INT8 uniform

Prompt: Explain quantum computing briefly.
--------------------------------------------------

that operates using quantum mechanics rather than classical
computing principles...
--------------------------------------------------
Generated 80 tokens in 4.28s (18.7 tok/s)
